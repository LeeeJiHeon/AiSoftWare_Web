## 💻 프로젝트 소개
강원대학교 AI소프트웨어학과 홈페이지를 클론코딩 하였으며, 게시판 기능을 추가하였습니다.
<br><br>

## ⌚ 개발 기간
- 22.04.12 - 22.06.07
<br><br>

## ⚙️ 개발환경
- Eclipse (`Java 1.8`), JSP, JDK, Chrome
- Bootstrap, Popper, Jquery,
- Apache Tomcat(`9.0`)
- MySQL
<br><br>
